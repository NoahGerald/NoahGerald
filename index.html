Web Interface Design - GitHub Page- CONTENTS

The content of this module is structured into 6 Didactic Units:

1. Planning of graphical interfaces:

  • Design elements: visual perception.
  • Web interface components. Components — Material Design 3
  • Color. Theme creation tool. Color palettes Color – Material Design 3
  • Icons Icons – Material Design 3
  • Typography Typography – Material Design 3
  • Prototyping. Prototyping tools.
  • Style guide interpretation. Style guide creation. Style Guide - Component Catalog - Junta de Castilla y León
  • Navigation Maps.
  • Web layout. HTML 5 Markup Language.
  • Pattern detection.
  • Design templates. Material/Bootstrap.

2. Use of styles:

  • W3C recommendations.
  • Inline styles based on tags, classes, and identifiers.
  • Create and link style sheets. Create and link external cascading style sheets.
  • Styles for typography and interface component appearance.
  • Box model.
  • Styles for positioning and displaying page elements.
  • Distribution structures (Layouts). Flex/grid.
  • Responsive design. Media Queries.
  • Design framework: MATERIAL/BOOTSTRAP.
  • Preprocessors: SASS (Bootstrap/Material).

3. Implementation of multimedia content:

Since this unit overlaps with the Web Development in Client-Side Environment module, in agreement with the professor teaching that module and in order to avoid repeating the same content, this unit will not be covered, and the hours dedicated to Units 1 and 2 will be increased, as they are considered more relevant to the module.

  • Intellectual property rights. Licenses. Copyright law. Copyright.
  • Images: Bitmap image formats, vector images. Software for creating and processing images. Image optimization for the Web. Size and resolution. Adaptive images.
  • Audio: Formats. Format conversions (export and import).
  • Video: Video encoding, format conversions (export and import). Video for the web. Adaptive video.
  • Animations: Animation of images and text. Integration of audio and video into animation.

4. Integration of interactive content:

(Due to the fact that this unit overlaps with the module of Web Development in Client Environment, in consensus with the teacher who teaches the module and in order not to repeat the same contents, this unit will not be seen and the hours dedicated to Unit 1 and 2 will be extended as they are considered to be more relevant to the module).

  • Basic and advanced interactive elements.
  • Interactive behaviors. Element behavior.
  • Execution of scripts.
  • Checking interactive content from different browsers.

5. Accessible web design:

  • Accessibility concept.
  • Recommendations from the World Wide Web Consortium (W3C).
  • Web Content Accessibility Guidelines (WCAG).
  • General principles of accessible design.
  • Techniques to meet the requirements defined in the WCAG:
    • Fundamental techniques.
    • HTML techniques.
    • CSS techniques.
  • Methods for conducting preliminary reviews and suitability evaluations.
  • Browser tools for accessibility.
  • Web accessibility analysis tools.
  • Checking web accessibility from different browsers.

6. Web usability implementation. User-friendly design:

  • Usability concept.
  • Web usability principles.
  • Usability implementation on the web.
  • Web usability tools.
